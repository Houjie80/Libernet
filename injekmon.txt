
Cara Setting Injekmon di Libernet Anti Bengong

1.Khusus Android Install Mobile SFTP
2.Khusus PC / Laptop Install WinSCP
3.Download PuTTY Khusus PC / Laptop
4.Download Termux Atau JuiceSSH Khusus Android


Install Injekmon
https://github.com/laksa19/openwrt-tools/tree/main/injekmon

Buka & Edit
/etc/rc.local

Buka & Edit
/root/injekmon.txt

Buka & Edit
/usr/bin/injekmon
/usr/bin/serviceinjekmon

Ganti PingLoop

cd root/libernet/ping-loop.sh << Cara & Edit Tulisan dibawah

INTERVAL="1"
HOST="IP AKUN"


Buka Putty Khusus Windows & Termux Khusus Android

injekmon ip Akun Injeksi

Buka PuTTY

Ketik > killall -q serviceinjekmon
