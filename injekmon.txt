Cara Setting Libernet Anti Bengong ( IP WAN GAK MUNCUL )

Setting Libernet Mod


1. Login Libernet
   http://192.168.1.1/libernet/index.php
   
   User     : admin  
   Password : libernet
   
2. Masuk Ke Folder Libernet
   root/injekmon.txt << edit & isi IP Akunya
   libernet/bin/ping-loop.sh << edit & isi IP Akunya
   
   Cara Tulisan ini
   
   INTERVAL="1"
   HOST="IP AKUN"
   
3. Buka PuTTY / Termux Khusus Android
   Ketik di bawah 
   killall -q serviceinjekmon
